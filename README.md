# Portfolio Template

---

This portfolio template is designed to be simple and flexible to anyone's needs.
Easy to install, configure, and maintain.

## Installation

---

### Step 1: Installing dependencies

- Run `npm install` and all dependencies that the project uses should be installed. 

Dependencies used:
- shadcn `npm install shadcn`


### Step 2: Running the server

- Run `npm run dev` to be able to run the server.
- Navigate to `localhost:3000` to view the content.
